About This Website

This website is a documentation of my time at LeWagon's Lisbon Batch 86 Bootcamp. I created this when we started to study HTML and CSS. This website is not only a documentation of some movies introduced to me or the food I like in Lisbon, but also a documentation of my education progress. While I have learned Bootstrap, this website does not use it, because I made so many of my own CSS classes that I want to keep here, as a memory. I have cleaned up and organized the CSS a little, but I hope I will never be so messy in my code ever again. This is evidence of where I started making websites and a collage of my experience in Portugal.
